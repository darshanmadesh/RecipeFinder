# RecipeFinder
An app to find, create, update and delete recipes.

## Run the App
1. run npm install in the main folder
2. go into the seeds folder and run index.js
3. once completed come back to main folder and run app.js

## Packages used:
### Node
### Express
### MongoDB
### Mongoose
### ejs
### ejs-mate
### method-override
### nodemon
